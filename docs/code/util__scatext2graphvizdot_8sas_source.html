dotMySASProgram
Custom built SAS macro programs to trigger recording and extracting information about input, output, and the use of macro symbols from a SAS program while it is running. The captured information can provide a simple static data lineage diagram. As a minimum, Base SAS Software, version 9.4 Maintenance 4 installation is required to run these macros.

1/**
2 @file util_scatext2graphvizdot.sas
3 @brief Processes a list of scaProc generated text files.
4 @details Accepts a list a scaProc (SAS Code Analyzer) generated text files, and dynamically
5 composes the required %util_parseScaproc macro call for every file in the list, before executing
6 all the generated calls to mass convert/parse sca text to a GraphVis *.dot file
7
8 Example usage:
9
10 %util_scaText2graphvizDot (

13 , p_includeAttrs_yn=N
14 , p_includeSteps_yn=Y
15 , p_rankDir=LR)
16
17 @param [in] p_inTxtFileList= File name containing list of scaProc (SAS Code Analyzer)
18 generated text file names. Required.
19 @param [in] p_outDotFilesDir= The GRAPHVIZ (*.dot) output directory. Required.
20 @param [in] p_includeAttrs_yn= A Y/N flag to include table columns in the output. Default:Y.
21 @param [in] p_includeSteps_yn= A Y/N flag to include processing steps and elapse times in the output.
